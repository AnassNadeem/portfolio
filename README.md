# APEX — Anas Nadeem · F1-Grade Portfolio

A scroll-driven, racing-themed, **playable** portfolio. React 19 + TypeScript + Vite,
**Three.js** (React Three Fiber), **GSAP** (ScrollTrigger, MotionPath, DrawSVG, SplitText,
ScrambleText), **Framer Motion**, **Lenis** smooth scrolling, synthesized **WebAudio**.

Zero image assets: the car is procedural geometry, decals are canvas textures, circuit
art is generated SVG, engine sounds are synthesized, the resume PDF is generated by a
script, and the OG card is a real screenshot of the hero.

## Run it

```bash
npm install
npm run dev        # → http://localhost:5173
npm run build      # regenerates resume.pdf + production build → dist/
npm run preview
```

## The experience

| System | What it does |
| --- | --- |
| **Reaction-time start** | The preloader ends with the real F1 start: five lights, random hold, lights out — click. Your reaction time is scored, verdicted, and posted to the arcade leaderboard. Jump it and it's a false start. Fully skippable ("SKIP FORMATION LAP"); returning visitors get a short load. |
| **3D hero garage** | Procedural F1 car #42 with **ANAS NADEEM** decals, mouse steering, rev (click), tire smoke, synthesized V6. **BOX BOX** runs a full pit stop — jack up, wheels off, compound swap, 2.x s on the clock. Rain light blinks when you scroll *up* (braking); exhaust glows when you scroll down (throttle). Onboard halo-cam toggle in the HUD. |
| **The Garage (Sector 02)** | The signature section: scroll pins the scene and the car **explodes into its parts — each one mapped to a real skill domain** (front wing = frontend, power unit = AI & data, halo = testing, floor = infra…). Tap any floating label for its spec sheet. The machine is the resume. |
| **Racing line** | A winding track through every section; a top-down car scrubbed by scroll lays twin tire marks. The **DRS zone is real** — scroll fast through Sector 2 and the flap opens with a toast. |
| **Telemetry HUD** | Circular rev gauge with needle + gear, digital speed, and a **live minimap of the actual racing line that fills with color as you cover distance**. SND / CAM / ARCADE controls. |
| **Hot lap + sectors** | The whole site is a timed circuit: three sector splits (green = close, purple = personal best) and a chequered line at the footer. Finish → **podium overlay**: confetti, lap time, *Champion Gold livery unlock*, and a "Sign Me For Your Team" CTA. |
| **Arcade (leaderboards)** | RACE CONTROL modal with three games — **Lights Out** (reaction), **Pit Stop** (hit all four wheel guns), **Hot Lap** (the scroll itself) — plus per-game leaderboards with seeded BOT rivals, arcade initials and optional email. Device-local by default; see "Real leaderboard" below to make it global. |
| **Pit Wall (projects)** | Mission-control screen bank from the future: CRT power-on flicker, scanlines, sweep refresh, tilted monitors, live UTC clock — and **live GitHub telemetry per repo** (stars, forks, language, last push) straight from the public API, cached 6h. |
| **Live season stats** | About section pulls real GitHub profile numbers (repos, followers, top languages). |
| **⌘K Race Control** | Command palette: sectors, actions (rev, box box, onboard cam, hot lap, flying lap, arcade, sound), liveries, links. Also opens with `/`. |
| **Cheat codes** | Type anywhere: `drs` (8s overtake mode), `boxbox` (pit stop), `onboard` (halo cam), `gold` (champion livery). Console has ASCII art + hints for DevTools visitors. |
| **30-sec hot lap tour** | "Auto tour" chip in the hero: cinematic auto-scroll with race-engineer radio captions. Any input hands control back. Ends on the finish line → podium. |
| **Livery picker** | Rosso / Papaya / Petronas / Oracle (+ unlockable Champion Gold) — repaints the 3D car AND re-themes the entire site live. |
| **Tire cursor, magnetic buttons, scramble nav** | The cursor is a tire that rolls with travel; buttons lean toward the pointer; nav links de-scramble on hover. |
| **Print stylesheet** | ⌘P prints a clean, typographic resume page — no theatrics. |
| **404 — DNF** | `public/404.html`: car in the gravel trap, "Return to the pits". |

Accessibility: `prefers-reduced-motion` collapses everything to simple fades (no preloader
game, no racing line, no HUD); keyboard reachable; palette and arcade are dialogs with ESC.

Performance: the entire Three.js stack is **code-split** (`F1Car` chunk lazy-loads behind
the preloader; main bundle ≈ 205 KB gzip). Both canvases pause rendering when offscreen.

## Real (global) leaderboard — optional

Out of the box scores are per-device with BOT rivals. To make it global, deploy this
Cloudflare Worker (free tier) with a KV namespace bound as `LB`, then paste the URL into
`LEADERBOARD_ENDPOINT` in `portfolio.ts`:

```js
export default {
  async fetch(req, env) {
    const cors = {
      "Access-Control-Allow-Origin": "*",
      "Access-Control-Allow-Headers": "Content-Type",
      "Access-Control-Allow-Methods": "GET,POST,OPTIONS",
    };
    if (req.method === "OPTIONS") return new Response(null, { headers: cors });
    const game = new URL(req.url).pathname.slice(1); // reaction | pitstop | hotlap
    if (!["reaction", "pitstop", "hotlap"].includes(game))
      return new Response("nope", { status: 404, headers: cors });
    const list = JSON.parse((await env.LB.get(game)) ?? "[]");
    if (req.method === "POST") {
      const { name, ms, email } = await req.json();
      if (typeof ms !== "number" || ms < 50 || ms > 600000)
        return new Response("invalid", { status: 400, headers: cors });
      list.push({ name: String(name).slice(0, 3).toUpperCase(), ms: Math.round(ms), email, date: new Date().toISOString().slice(0, 10) });
      list.sort((a, b) => a.ms - b.ms);
      await env.LB.put(game, JSON.stringify(list.slice(0, 100)));
    }
    return Response.json(list.map(({ email, ...e }) => e), { headers: cors });
  },
};
```

## Deploy

Static site — Vercel / Netlify: framework "Vite", build `npm run build`, output `dist`.
`public/404.html` is picked up automatically by Netlify and GitHub Pages.
`og.png` is referenced relative (`/og.png`) — most platforms resolve it; for strict
crawlers set an absolute URL in `index.html` once you know your domain.

## Structure

```
src/
  data/portfolio.ts          ← ALL content: identity, projects, garage map, bots, tour
  context/AppContext.tsx     ← Lenis, accent/livery, camera mode, modals, velocity
  lib/
    bus.ts                   ← typed event bus (drs, sector, finish, pitstop, hotlap…)
    github.ts                ← live GitHub stats (cached 6h)
    leaderboard.ts           ← local + bots + optional remote adapter
    sound.ts                 ← synthesized engine / shift blips
  components/
    Preloader.tsx            ← tire-roll loader + reaction-time start game
    RacingLine.tsx           ← scroll-scrubbed car + tire marks + DRS zone
    SpeedHUD.tsx             ← rev gauge + live track minimap + race toasts
    RaceTimer.tsx            ← lap timing, sector splits, finish detection
    Podium.tsx               ← confetti finale + gold unlock + CTA
    GameCenter.tsx           ← arcade: 3 games + leaderboards
    CommandPalette.tsx       ← ⌘K race control
    HotLapTour.tsx           ← 30s auto-scroll with radio captions
    EasterEggs.tsx           ← cheat codes, console art, magnetic buttons
    three/F1Car.tsx          ← procedural car, exploding parts, decals, pit stop
    three/HeroScene.tsx      ← showroom (TV + onboard cams)
    three/GarageScene.tsx    ← exploded view + floating part labels
    sections/                ← Hero, About, Garage, Experience, Projects (Pit Wall),
                               Skills, Contact, Footer (+ TeamRadio strip)
scripts/make-resume.mjs      ← regenerates public/resume.pdf on every build
public/404.html              ← DNF page
```

> Tip: `node_modules` inside OneDrive can sync-thrash — exclude it from sync if installs feel slow.
